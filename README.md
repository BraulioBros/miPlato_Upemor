Pasos para agregar la base de datos a PHPMYADMIN
PASOS
1. Crear una base de datos con el mismo nombre (miPlato_Upemor)
2. Ir al apartado de "importar"
3. Seleccionar el archivo sql previamente descargado de la carpeta sql de este repositorio.
4. Listo, si todo sale bien la base de datos deberia ya de estar cargada y lista para su uso.
